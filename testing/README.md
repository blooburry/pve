# Python Testing Tool

## Dependencies

- [paho-mqtt](https://pypi.org/project/paho-mqtt/)

Je kan deze library installeren met pip:

```bash
$ pip install paho-mqtt
```

Er moet ook een MQTT broker runnen op `localhost:1883`.
